# 2026 웹디자인개발기능사 실기 연습

웹디자인개발기능사 실기 시험을 준비하면서 정리한 연습 기록 저장소입니다.  
유형별 실습 결과물, 학습 메모, 참고 링크를 한곳에 모아두고 반복 학습할 수 있도록 구성했습니다.

## 목적

- 실기 시험 전 범위를 유형별로 정리하며 체계적으로 연습하기
- 레이아웃 구조와 구현 패턴을 직접 만들며 익히기
- HTML/CSS/JS 작성 속도와 정확도 높이기
- 실습 중 배운 점과 주의사항을 기록해 재학습하기
- 시험 직전 빠르게 복습할 수 있는 개인용 자료집 만들기

## 웹디자인개발기능사 실기 개요

웹디자인개발기능사(구 웹디자인기능사)는 웹디자인 및 웹퍼블리싱의 기본 역량(HTML/CSS/JS)을 평가하는 국가기술자격입니다.

실기에서는 주어진 시안과 요구사항에 맞춰 웹 페이지를 구현하는 능력이 중요하며, 레이아웃 구성과 마크업 정확도가 핵심입니다.

## 저장소 구성

```text
.
├── README.md
├── A/
│   ├── A-1/
│   ├── A-2/
│   ├── A-3/
│   └── A-4/
├── B/
├── C/
├── ...
├── S/
├── 강의메모&팁/
├── 관련링크/
└── 공개문제/
```

### 폴더 설명

- `A/`
  - 현재 정리 중인 유형 실습 폴더
  - 각 하위 폴더에 HTML 결과물, 예제 레이아웃 이미지, 구현 미리보기 이미지 포함
- `B/`, `C/`, `...`, `S/`
  - 추후 순차적으로 추가할 유형별 실습 폴더
  - 최종적으로 `A`부터 `S`까지 실기 연습 자료를 확장해 정리할 예정
- `강의메모&팁/`
  - 실습 중 정리한 메모, 주의사항, 시험 대비 팁 정리
- `관련링크/`
  - 자격증 정보, 강의 링크, 피그마 레퍼런스 링크 정리
- `공개문제/`
  - 공개문제 PDF 등 시험 대비 참고 자료 보관

## 현재 학습 범위

현재는 `A` 유형 일부 실습 자료가 먼저 정리되어 있으며, 이후 `B`부터 `S`까지 범위를 넓혀갈 예정입니다.

- `A/A-1/layout_A-1.html`
- `A/A-2/layout_A-2.html`
- `A/A-3/layout_A-3.html`
- `A/A-4/layout_A-4.html`

필요할 때 각 폴더의 이미지 자료와 함께 비교하면서 복습할 수 있도록 구성했으며, 저장소 범위는 점차 전체 유형으로 확장할 계획입니다.

## 학습 방식

이 저장소는 단순 결과물 보관용이 아니라, 실기 전 범위를 반복 연습하기 위한 개인 학습 로그 성격도 함께 갖고 있습니다.

- 예제 레이아웃 이미지를 보고 직접 구조 분석
- `div` 중심으로 기본 레이아웃부터 빠르게 작성
- 파일명, class명, id명, 주석 정리를 통해 실수 줄이기
- 학습 후 배운 점을 별도 메모 파일에 기록

예를 들어 `A/A-1/학습내용요약.md`에는 초기 HTML 작성, 자동완성 활용, 네이밍, 가운데 정렬 같은 기본기를 짧게 정리해두었습니다.
