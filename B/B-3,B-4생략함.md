# B-3, B-4 정리 생략 메모

## B 유형 핵심 특징

B 유형은 `header`와 `footer`의 배경이 화면 전체 너비를 차지하고, 실제 내부 콘텐츠만 `max-width`로 가운데 정렬되는 구조가 핵심입니다.

즉, 겉 배경은 `width: 100%`로 넓게 깔고, 안쪽 콘텐츠 박스만 제한 폭으로 관리하는 형태입니다.

## 현재 학습 판단

`B-3`, `B-4`는 세부 콘텐츠 배치 차이만 조금 있을 뿐, 큰 틀에서는 `B-1`, `B-2`와 같은 구조로 판단했습니다.

- 상단과 하단은 전체 폭 배경 처리
- 내부 콘텐츠는 가운데 정렬
- 메인 영역에서만 섹션 구성 차이 발생

구조 이해와 구현 방식이 이미 충분히 익숙해진 상태라, 두 유형 모두 혼자서 약 12분 내 구현 가능한 것을 확인했습니다.

## 이번에 정리 생략한 이유

남은 실습 분량이 많기 때문에, 이미 충분히 반복되어 확실하게 구현 가능한 유형은 상세 기록을 생략하고 넘어가기로 했습니다.

이번 `B-3`, `B-4` 생략은 대충 넘긴 것이 아니라 아래 기준을 충족했기 때문입니다.

- 공통 레이아웃 구조를 이미 이해함
- 전체 폭 배경 + 내부 `max-width` 구조를 직접 구현 가능함
- 제한 시간 안에 반복 구현이 가능함

## 다시 확인할 포인트

나중에 다시 볼 때는 아래만 빠르게 점검하면 됩니다.

- `header`, `footer`가 부모 `max-width`에 갇히지 않는지
- 내부 콘텐츠 래퍼가 가운데 정렬되는지
- 메인 영역 섹션 비율만 문제 요구사항에 맞게 바꾸면 되는지

## 메모

앞으로도 학습량이 많을 때는, 이미 충분히 숙달된 유형은 이런 방식으로 간단히 기록만 남기고 다음 유형으로 넘어갈 예정입니다.
